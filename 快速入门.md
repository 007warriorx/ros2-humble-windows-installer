# ROS2 Humble Windows 快速入门指南

## 5分钟快速开始

### 1. 安装（1分钟）
双击运行 `1_安装ROS2.bat`，等待安装完成。

### 2. 启动（30秒）
双击运行 `2_启动ROS2环境.bat`，会打开一个命令行窗口。

### 3. 测试通信（3分钟）

**需要同时打开两个窗口**，都运行 `2_启动ROS2环境.bat`：

**窗口1** - 输入：
```bash
ros2 run demo_nodes_py talker
```

**窗口2** - 输入：
```bash
ros2 run demo_nodes_py listener
```

如果窗口2显示收到消息，说明ROS2工作正常！

按 `Ctrl+C` 停止节点。

## 常用命令速查

| 命令 | 说明 |
|------|------|
| `ros2 topic list` | 列出所有话题 |
| `ros2 node list` | 列出所有节点 |
| `ros2 pkg list` | 列出所有包 |
| `ros2 doctor` | 诊断ROS2状态 |
| `ros2 run demo_nodes_py talker` | 运行发布者示例 |
| `ros2 run demo_nodes_py listener` | 运行订阅者示例 |

## 注意事项

1. **每次使用ROS2前**，必须先运行 `2_启动ROS2环境.bat`
2. Python必须是 **3.10版本**，其他版本不兼容
3. 安装前确保已安装 **Visual C++ Redistributables**
4. 详细问题排查请参考 `README.md`
