四、安装后目录 (自动生成)
-------------------------
C:\dev\ros2_humble
├── bin\                   - DLL和可执行文件
├── Lib\                   - Python库
│   ├── site-packages\     - Python包
│   ├── demo_nodes_py\     - Python示例节点
│   └── demo_nodes_cpp\    - C++示例节点
├── Scripts\               - Python脚本（ros2命令）
├── share\                 - 包数据和资源
├── include\               - C++头文件
└── cmake\                 - CMake配置

五、系统要求
------------
- 操作系统: Windows 10/11 64位
- Python版本: 3.10.x (必须)
- 内存: 4GB以上（推荐）
- 磁盘空间: 5GB以上

六、前置依赖（用户自行安装）
---------------------------
1. Python 3.10
   下载: https://www.python.org/downloads/release/python-31011/

2. Visual C++ Redistributables
   下载: https://aka.ms/vs/17/release/vc_redist.x64.exe

3. ROS2安装包
   下载: https://github.com/ros2/ros2/releases
   文件: ros2-humble-*-windows-release-amd64.zip

------------

八、使用流程
------------
1. 安装Python 3.10和VC++ Redistributables
2. 下载ROS2安装包放入"已下载软件"目录
3. 运行 1_安装ROS2.bat
4. 运行 2_启动ROS2环境.bat 启动ROS2
5. 运行 3_测试ROS2.bat 测试安装

十、注意事项
------------
1. Python必须是3.10版本
2. 安装时请勾选"Add Python to PATH"
3. 每次使用ROS2前需运行启动脚本
4. 如遇到问题请查看README.md故障排除部分
